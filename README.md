#  Laptop Price Analysis Using Python

---

##  Overview
This project explores how various hardware and software features affect **laptop prices** using Python.  
It includes a comprehensive **Exploratory Data Analysis (EDA)** to identify market trends, pricing drivers, and customer preferences across brands and configurations.

---

##  Objective
1. Analyze how specifications such as **RAM, CPU, GPU, storage type, and display** influence laptop prices.  
2. Identify **key factors** driving laptop costs.  
3. Compare **pricing trends across brands** and product types.  
4. Provide **data-driven insights** for manufacturers, retailers, and consumers.

---

##  Data Preprocessing
### Steps Performed
1. Loaded and explored the dataset.  
2. Checked data shape and column overview.  
3. Removed duplicates and handled missing values.  
4. Converted data types and standardized features.  
5. Conducted descriptive statistics to understand data distribution.

##  Key Insights
1. **Performance drives price:** High-end CPUs, GPUs, SSDs, and large RAM significantly increase cost.  
2. **SSDs dominate** modern laptops for speed and reliability; HDDs remain in budget models.  
3. **4K/Retina displays** and premium build quality command higher prices.  
4. **Dell, Lenovo, and HP** lead in both sales and revenue; **Apple** dominates the premium market.  
5. **Students (98.5%)** are the main buyers, preferring mid-range laptops (8‚Äì16 GB RAM, SSD, i5/i7 CPUs).

---

##  Conclusion
Laptop pricing is primarily influenced by:  
- **RAM and CPU performance**  
- **GPU model (NVIDIA/Intel)**  
- **Storage type (SSD > HDD)**  
- **Display quality (4K, IPS, Retina)**  
- **Brand reputation and design quality**
